# Travel and Tourism Management System

## 📌 Project Description
This project is a web-based Travel and Tourism Management System that allows users to explore destinations, view packages, and book trips online.

## 🎯 Features
- Destination browsing
- Travel packages
- Booking system
- User login
- Admin panel

## 🛠 Technologies Used
- HTML
- CSS
- JavaScript

## 📂 Project Structure

- destinations.html
- booking.html
- css/
- js/
